Assignment 2: Responsive Redesign

Description

In this assignment, my task was to select and redesign one page from any website that showed clear faults in design based on four main criteria of good design: learnability, memorability, efficiency and accessibility.
Once chosen, I was expected to create three low fidelity sketches, three high fidelity designs and a visual design guide, each a step towards creating an improved version of the original website. Throughout the annotations in my designs, I address how I was able to solve the problems identified with the page and I provide brief notes on how to implement the page on your own. The final design was constructed using HTML, CSS and some JavaScript.

Part 1: Identifying Problems

Description

This website provides information for a Providence-based cafe chain (three locations, only one is shown), including a menu, details on locations, and the option to contact via a form.

Original website + Screenshot

http://bagelgourmetcafe.com/

Summary of Problems

Criteria Issues Identified
Learnability
  • Inconsistent sizing and layout of images on front page can be disorienting for new users
  • Below the navigation bar, there are instances of bold, underlined text that can be confused for links
Memorability
  • Users may be confused with navigation even after frequent visits, given that there are two identical sets of navigation tabs at the top and bottom of the screen
Efficiency
  • Small fonts on the navigation bars make it hard to click on links to different pages
  • Information located on the side bar is difficult to read, meaning that some users may repeatedly need to go to “Location” page instead
  • Page is made to fit computer screens but does not resize to fit tablet or phone screens

WAIVE Errors My Opinion
Alternate text is null or empty Agree:
... Images should have an alt text for visually impaired users
Empty heading Somewhat Agree:
... This does not affect the web page, but it may indicate a flaw within the HTML file
Low color contrast Agree:
... Users who are visually impaired may have trouble reading the text clearly
Missing or uninformative page title Somewhat Agree:
... The text font and color for the store name blends in with the background image
